# Logic

Here I will collect information and packages related to logic and more
specifically to description logics (DL) and inference engines.


## Highly relevant packages

 * [rdf4h](https://github.com/robstewart57/rdf4h) - package for parsing RDF.
   [Solid documentation](http://robstewart57.github.io/rdf4h/).

 * [hsparql](https://github.com/robstewart57/hsparql) - Build and submit
   queries to a SPARQL database. Updated within the last 7 months with solid
   examples in github. Written by the author of `rdf4h`.

 * [swish](https://hackage.haskell.org/package/swish) - collection of tools for
   inference on RDF.
   [Documentation](http://www.ninebynine.org/RDFNotes/Swish/Intro.html). It has
   a [Notation3](https://www.w3.org/DesignIssues/Notation3.html) parser. N3
   allows expression of rules in the RDF framework.


## Slightly off-topic logic stuff

 * [Folly](https://github.com/dillonhuff/Folly) - First Order Logic library
   (last update, 2 years ago).

 * [WeberLogic](https://github.com/cameronbwhite/WeberLogic) - Parse
   propositional logic expressions into abstract syntax trees.

 * [hol](https://github.com/gilith/hol) - Read proof files in higher-order
   logic using [OpenTheory](http://www.gilith.com/opentheory/) format.

 * [Logic](http://hackage.haskell.org/package/Logic) - Very simple L0

 * [helf](https://github.com/andreasabel/helf) - A partial implementation of
   [Twelf](http://twelf.org/wiki/Main_Page)

 * [haskhol-core](http://ecaustin.github.io/haskhol/) - A domain-specific language for HOL in haskell

 * [Jukebox](http://hackage.haskell.org/package/jukebox) - Jukebox is a suite
   of tools for transforming problems in first-order logic. It reads problems
   in [TPTP](http://www.cs.miami.edu/~tptp/) (FOF and TFF) format.


## Arcana

 * [tersmu](http://hackage.haskell.org/package/tersmu) - A semantic parser for
   (Lojban)[https://mw.lojban.org/papri/Lojban], a logic-based conlang.
